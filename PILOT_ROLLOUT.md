# Pilot Rollout

Use this when sending DodoLaunch India to the first 20 users, then the next 200.

## First 20 Users

Send the live Vercel app and ask each tester to complete one flow:

1. Open the app on desktop or mobile.
2. Connect Phantom from the top-right wallet action.
3. Create a Dodo checkout.
4. Record the successful sale.
5. Review the revenue ledger.
6. Prepare the mainnet settlement batch.
7. Stop before approving a real transfer unless the wallet is intentionally funded.
8. Export the split CSV.
9. Leave feedback in GitHub Issues.

## What To Ask

- Did the product make sense in the first 20 seconds?
- Was wallet connect obvious?
- Did the checkout, ledger, and settlement flow feel connected?
- Was anything unclear before the mainnet approval step?
- Did the mobile layout feel usable?

## Fix Before Scaling To 200

- Broken wallet connect path.
- Mobile layout issues.
- Confusing mainnet approval copy.
- CSV export errors.
- Any path that suggests fake volume or fake chain proof.

## Submission Evidence

- GitHub repo link.
- Live Vercel link.
- Two-minute walkthrough video.
- Screenshot of the wallet-first settlement screen.
- Screenshot of the Dodo checkout path.
- Screenshot of the exported split CSV.
